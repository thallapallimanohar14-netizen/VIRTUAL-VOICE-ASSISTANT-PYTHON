
# 🎙️ Virtual Voice Assistant using Python

A smart, Python-based Voice Assistant that understands voice commands, responds using text-to-speech, performs web searches, tells the current time and date, and opens popular websites. This project demonstrates the fundamentals of speech recognition, voice interaction, and task automation using Python.

---

## 📖 Overview

The Virtual Voice Assistant is designed to make everyday computer tasks easier through voice commands. It listens to the user's speech, processes the command, and performs the requested action while providing voice feedback.

---

## ✨ Features

- 🎤 Speech Recognition using microphone input
- 🔊 Text-to-Speech (TTS) responses
- 🌐 Google Search through voice commands
- 🕒 Announces current time
- 📅 Announces current date
- 🌍 Opens websites like Google, YouTube, and more
- 💬 Interactive voice communication
- ⚡ Fast and easy to use

---

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/virtual-voice-assistant-python.git
```

### Move into the project folder

```bash
cd virtual-voice-assistant-python
```

### Install dependencies

```bash
pip install SpeechRecognition pyttsx3 PyAudio wikipedia
```

### Run the project

```bash
python Voice_Assistant.py
```

## 💡 How It Works

1. Starts the voice assistant.
2. Listens to the user's voice through the microphone.
3. Converts speech into text.
4. Identifies the requested command.
5. Executes the command.
6. Responds using text-to-speech.
